# JetBrains IDE Support

The [GraphQL Plugin for JetBrains](https://plugins.jetbrains.com/plugin/8097-graphql/) supports Apollo Federation, but it is disabled by default. To enable it:

1. Open the "Settings" window
2. Expand the "Languages & Frameworks" section
3. Click on "GraphQL"
4. Check the "Federation" checkbox

This plugin supports all the IntelliJ-based IDEs, such as:

- IntelliJ IDEA
- PyCharm
- PhpStorm
- WebStorm
- CLion
- RubyMine
- Rider
- GoLand
